# Contributing to Debian Packaging Mentor

Thank you for your interest in contributing!

## Ways you can help

- Suggest improvements to prompts or behavior
- Report incorrect packaging guidance
- Share real-world packaging cases for testing
- Improve documentation
- Propose new features
- Help test different packaging workflows

## How to submit an issue

Please include:

- What you were trying to do
- Example project or packaging files
- Any `lintian` output
- Expected behavior
- Actual behavior

## Pull Requests

Please:

1. Keep changes focused and clear
2. Explain your reasoning
3. Reference Debian Policy sections when relevant
4. Be respectful - this project exists to help learners

Thank you!
